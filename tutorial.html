Tutorial: Guida al Desktop

1. File sul Desktop

Le icone sul desktop rappresentano file (PDF) o cartelle. Clicca una volta per aprirli. I PDF si apriranno in una finestra dedicata al centro dello schermo.

2. Gestione Finestre

Ogni finestra ha due controlli in alto a sinistra:

  • X (Chiudi): Chiude definitivamente la finestra.
  • - (Riduci): Minimizza la finestra nel Dock in basso.

3. Il Dock

Il Dock in basso raccoglie tutte le finestre minimizzate. Clicca sull'icona nel Dock per ripristinare la finestra alla sua dimensione originale.

4. Ridimensionamento

Dal menu "Ridimensiona" in alto, puoi regolare la grandezza delle icone del desktop. Il sistema riorganizzerà automaticamente le icone per evitare sovrapposizioni.

5. Account e Persistenza

Per salvare le tue modifiche (come la dimensione delle icone o gli appunti):

  • Apri il menu Account.
  • Inserisci una mail e una password per creare il tuo profilo.
  • Una volta loggato, tutte le modifiche verranno salvate e ricaricate al tuo prossimo accesso.
  • Senza account, il sito tornerà allo stato iniziale ogni volta che chiudi il browser.

6. Appunti

Usa il menu "Appunti" per scrivere note veloci. Se sei loggato, i tuoi appunti verranno salvati nel tuo profilo.
