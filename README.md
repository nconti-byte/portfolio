# Portfolio Personale "Web Desktop"
Un sito web portfolio unico che simula l'esperienza di un desktop moderno.

## Obiettivo
Il sito rappresenta l'identità professionale dello sviluppatore attraverso una metafora visiva familiare: un desktop con icone, cartelle e finestre interattive.

## Stack Tecnologico
- **HTML5**: Struttura semantica e contenitori per il desktop.
- **CSS3**: Glassmorphism, layout absolute/fixed, animazioni e responsività.
- **JavaScript Vanilla**: Gestione delle finestre (apertura, chiusura, trascinamento) e orologio di sistema.
- **Lucide Icons**: Set di icone moderno e pulito.
- **Google Fonts**: Montserrat (Titoli) e Open Sans (Testo).

## Struttura delle Pagine
- `index.html`: Home page con una breve introduzione e call-to-action.
- `bio.html`: Pagina biografia con percorso formativo e professionale.
- `portfolio.html`: Galleria dei progetti realizzati.
- `lavoro-1.html`: Dettaglio approfondito del primo progetto.
- `lavoro-2.html`: Dettaglio approfondito del secondo progetto.
- `lavoro-3.html`: Dettaglio approfondito del terzo progetto.
- `contatti.html`: Pagina con informazioni di contatto e form.

## Struttura delle Cartelle
- `/css`: Contiene `style.css` per la gestione di tutti gli stili del sito.
- `/img`: Destinata a contenere tutte le risorse grafiche e fotografiche.
- `/js`: Contiene `main.js` per gli script di interattività globale.

## Comandi Utili
Per visualizzare il progetto in locale:
1. Aprire il file `index.html` direttamente in un browser.
2. (Consigliato) Utilizzare l'estensione **Live Server** in VS Code per ricaricamento automatico.
